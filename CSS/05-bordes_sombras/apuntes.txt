Las propiedades del borde CSS le permiten especificar el estilo, el ancho y el color del borde de un elemento.

Los shurthands son:
    * border-style: none (por defecto) | hidden | solid | dotted |dashed | double | groove | ridge | insert | outset
    * border-width: thin |medium | thick | medida
    * border-color: Hereda el color del elemento
